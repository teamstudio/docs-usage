## Teamstudio Usage Auditor について
Usage Auditor を使用することで、Notes アプリケーションのユーザー利用状況を追跡し、稼働中のデータベース及び IT 環境に関する、より明確な意思決定を可能にする利用状況トレンドを把握できます。

Notes アプリケーションをモニターすることで以下の効果が得られます:

* 時間の削減: 運用継続すべきアプリケーションを特定できる
* 環境に関する理解、特に引き継ぎなどで概要がよく掴めない問題をなくす
* コストの削減: 不要アプリを見つけサーバー数を減らすことで管理リソースを削減
